# TTGDT Shedule Bot

Бот, предназначенный для получения расписания/замен в Томском техникуме железнодорожного транспорта.

## Планы

- [X] Получение расписания по группам
- [ ] Получения замен по группам
- [ ] Настройки бота
- [ ] Парсинг данных в фоне
- [ ] **Inline-режим**
